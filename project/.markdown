## üéØ Overview

This application allows users to:
- **Extract transcripts** from any YouTube video
- **Auto-detect language** and translate to English
- **Generate AI summaries** of video content
- **Ask questions** and get context-aware answers

## üîß How It Works

### 1. Transcript Extraction

```python
# Extract video ID from URL
video_id = extract_video_id("https://www.youtube.com/watch?v=VIDEO_ID")

# Fetch transcript using YouTube Transcript API
ytt_api = YouTubeTranscriptApi()
transcript_list = ytt_api.list(video_id)
transcript_data = ytt_api.fetch(video_id, languages=['en', 'hi'])
```

### 2. Language Detection & Translation

```python
# Using Google Translate
from googletrans import Translator

translator = Translator()
detection = translator.detect(text[:500])

if detection.lang != 'en':
    translated = translator.translate(text, src=detection.lang, dest='en')
```

### 3. Text Chunking

```python
# Split text into chunks for better retrieval
splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50
)
chunks = splitter.split_documents(documents)
```

### 4. Vector Store Creation

```python
# Create embeddings and store in ChromaDB
embedding = HuggingFaceEmbeddings(model_name="sentence-transformers/all-MiniLM-L6-v2")

vector_store = Chroma.from_documents(
    documents=chunks,
    embedding=embedding,
    collection_name="youtube_transcript"
)
```

### 5. RAG Pipeline

```python
# Retrieve relevant chunks and generate answer
retriever = vector_store.as_retriever(search_kwargs={"k": 4})

chain = (
    {"context": retriever | format_docs, "question": RunnablePassthrough()}
    | qa_prompt
    | model
    | parser
)

answer = chain.invoke(question)
```
